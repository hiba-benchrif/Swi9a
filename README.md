# Swi9a

A simple MVC web application for Morocco price tracking and personal budgeting.

## Tech Stack
- FastAPI
- SQLite & SQLAlchemy
- Jinja2 Templates
- HTML & CSS (No JS)

## Running the Application
```bash
docker-compose up --build
```
Or locally:
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
